# oh-my-kimi Website 🌐

This repository contains the source code for the [oh-my-kimi Official Website](https://wang-h.github.io/oh-my-kimi-website/).

## Project Structure
`oh-my-kimi` is an orchestration layer for Kimi Code CLI, available in two specialized editions:

1.  **🐍 [Python Edition](https://github.com/wang-h/oh-my-kimi-python)**: The native Python orchestrator. Optimized for AI researchers and direct integration with Kimi's Python environment.
2.  **📦 [NPM Edition](https://github.com/wang-h/oh-my-kimi)**: The hybrid TypeScript/Rust orchestrator. Optimized for full-stack developers and Node.js toolchains.

## Website Features
- **Getting Started**: Clear installation paths for both editions.
- **Skill Catalog**: Documentation for shared OMK skills.
- **Compatibility Guides**: Support matrix for different platforms and Kimi CLI versions.

## Development
To run this website locally:
```bash
# Simple static server
python3 -m http.server 8000
```

## Deployment
Automated via GitHub Actions on push to `main`.
