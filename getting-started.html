Choose Your Edition

oh-my-kimi is available in two flavors. Choose the one that best fits your development environment.

🐍 Python Edition

Recommended. Best for AI researchers, algorithm engineers, and Python-native workflows.

1. Install: 
pip install oh-my-kimi-python
2. Initialize: 
omk setup
3. Run: 
omk team "Task description"

Deep integration with Kimi-CLI Python environment.

📦 NPM Edition

Best for full-stack developers and users who prefer Node.js toolchains.

1. Install: 
npm install -g oh-my-kimi
2. Initialize: 
omk setup
3. Run: 
omk

Leverages Rust-based core for high-performance IO.

Which one should I choose?

If you are working extensively with Python (Data Science, ML, AI scripts), the Python Edition allows you to write custom skills in pure Python. If you want a lightweight command wrapper within your existing npm-based workspace, the NPM Edition is for you.
