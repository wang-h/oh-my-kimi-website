Start here

Install the Python-native Kimi CLI orchestrator.

The new Python version provides deep integration with the Kimi CLI ecosystem and modern AI toolchains. Follow these steps to set up your environment and start orchestrating agents.

1. Install (Python Edition)

# Clone and install in editable mode
pip install oh-my-kimi-python

Install the core orchestration layer. Supports Python 3.10+.

2. Initialize Environment

omk setup

Configures ~/.kimi home, API keys, and MCP servers automatically.

3. Multi-Agent Team

omk team "Write a crawler"

Launches a Tmux session with one leader and three workers instantly.

4. Launch HUD

omk hud --watch

Monitor your agent activities and session metrics in real-time.

Legacy Migration Note

If you are upgrading from the npm version, simply uninstall the old package and alias omk to the new Python entry point. Your configurations in ~/.kimi are compatible. 

npm uninstall -g oh-my-kimi
alias omk='/path/to/omk-py'
