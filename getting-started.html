🐍 Python Native

The Orchestrator for AI Engineers

Deeply integrated with Kimi's native Python core. Ideal for those who want to write skills in pure Python and handle data-heavy workflows.

Quick Install
pip install oh-my-kimi

📦 NPM Hybrid

The Layer for Full-Stack Teams

A high-performance wrapper leveraging Rust for IO. Best for developers living in Node.js ecosystems who want a seamless CLI experience.

Quick Install
npm install -g oh-my-kimi

Next Steps after Installation

1. Initialize

Run omk setup to scaffold your ~/.kimi configuration and download essential skills.

2. Team Up

Try omk team "Task" to see multi-agent orchestration in action within your terminal.

3. Monitor

Launch omk hud --watch in a side pane to track agent progress and metrics in real-time.
