工作流快捷入口

技能，才是 oh-my-kimi 像“外壳产品”的地方。

这个项目不是单纯改个名字。技能把高频任务做成可重复的流程，让 Kimi CLI 更接近你熟悉的 oh-my-codex 风格使用方式。

技能作用为什么重要
omk setup安装提示词、技能、配置和本地状态骨架。把普通 Kimi CLI 安装变成一个可工作的 shell。
omk doctor检查安装、配置和运行状态。出问题时能快速知道 shell 哪一层坏了。
$deep-interview澄清范围、约束和缺失需求。让工作先变清楚，而不是直接猜着做。
$ralplan生成并审阅一个被批准的实施计划。让执行从“拍脑袋”变成“按计划推进”。
$ralph持续执行并带验证闭环。让任务更容易走到 done，而不只是停在建议层。
$team用 tmux worker 做协调并行。补上这种增强层最典型的多 lane 执行感。

规划

先用 $deep-interview 和 $ralplan 把任务理清，再进入执行。

执行

用 $ralph、$team、autopilot 把批准的工作往前推。

支持

用 doctor、hud、文档去理解这个 shell 的运行状态。
