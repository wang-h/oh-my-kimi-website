选择您的版本

oh-my-kimi 提供两个版本。请选择最适合您开发环境的一个。

🐍 Python 原生版

推荐。 适合 AI 研究员、算法工程师和习惯 Python 原生工作流的用户。

1. 安装: 
pip install oh-my-kimi-python
2. 初始化: 
omk setup
3. 运行: 
omk team "任务描述"

与 Kimi-CLI Python 环境深度集成，支持编写 Python 插件。

📦 NPM 混合版

适合全栈开发者和偏好 Node.js 工具链的用户。

1. 安装: 
npm install -g oh-my-kimi
2. 初始化: 
omk setup
3. 运行: 
omk

利用 Rust 核心提供高性能 IO 编排。

我该如何选择？

如果您的大部分工作涉及 Python（数据科学、机器学习、AI 脚本），Python 原生版 允许您用纯 Python 编写自定义技能（Skills）。如果您希望在现有的 npm 项目空间中获得一个轻量级的包装器，那么 NPM 混合版 更适合您。
