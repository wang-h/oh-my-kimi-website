安装 Python 原生 Kimi CLI 编排层

全新 Python 版本与 Kimi CLI 生态及现代 AI 工具链深度集成。 按照以下步骤配置环境并开始多 Agent 协作。

1. 安装 (Python 版)

# 克隆并以可编辑模式安装
pip install oh-my-kimi-python

安装核心编排层。支持 Python 3.10+。

2. 初始化环境

omk setup

自动配置 ~/.kimi 目录、API 密钥和 MCP 服务器。

3. 启动多 Agent 团队

omk team "写一个爬虫脚本"

立即拉起一个包含 1 个 Leader 和 3 个 Worker 的 Tmux 会话。

4. 开启 HUD 监控

omk hud --watch

实时监控 Agent 活动状态和会话指标。

旧版本迁移说明

如果您是从 npm 版本升级，只需卸载旧包并将 omk 别名指向新的 Python 入口即可。您在 ~/.kimi 中的配置是完全兼容的。 

npm uninstall -g oh-my-kimi
alias omk='/path/to/omk-py'
