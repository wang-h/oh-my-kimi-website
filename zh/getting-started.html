推荐

🐍 Python 原生版

面向 AI 工程师的编排器

与 Kimi 原生 Python 核心深度集成。适合需要用纯 Python 编写技能并处理重型数据工作流的用户。

快速安装
pip install oh-my-kimi

📦 NPM 混合版

全栈团队的增强层

利用 Rust 提升 IO 性能的高性能包装器。适合身处 Node.js 生态、追求无缝 CLI 体验的开发者。

快速安装
npm install -g oh-my-kimi

安装后的后续步骤

1. 初始化

运行 omk setup 来初始化 ~/.kimi 配置并下载基础技能。

2. 团队协作

尝试 omk team "任务"，在终端中即刻开启多 Agent 自动化编排。

3. 实时监控

在侧边窗口运行 omk hud --watch，实时追踪 Agent 进度与各项指标。
